C语言基础

switch

  • switch 仅能用于离散变量或值（整数值）
  • case 是分支入口，匹配成功后执行 case下的语句
  • 当语句全部执行后，需要使用 break跳出执行分支
  • default 分支不是必须的（类似于 else 不是必须的）

离散变量是指其取值限定在某个特定、可数的集合中的变量；这些取值通常是整数。
