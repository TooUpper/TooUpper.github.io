阿里云IOT

kay
2025-01-20
ESP32
ESP32-S3

阿里云 IoT（Internet of Things，物联网）是阿里巴巴集团旗下的云计算平台，专注于为物联网设备提供全面的解决方案，包括设备管理、数据采集、存储、分析和应用服务。

支持设备的快速接入，可以通过设备 SDK、MQTT、CoAP 等协议，方便设备与云端建立连接。

阿里云文档说明

阿里云IoT - 所知不止于感知

注册登陆后点击”文档”，在文档中心点击“阿里云物联网平台”，在左边的”操作指南”中点击”设备接入”，即可找到设备接入相关信息。这里我们使用的是”使用协议自主接入”。

注意要在 MQTT 协议中选择”MQTT-TLS 连结通信”

文档中详细的表明了接入需要的参数该如何设置

添加设备

在首页右上角点击”控制台”,进入后搜索”物联网平台”，在物联网平台中选择”公共实例”。

在公共示例左边点击”设备管理”，选择”产品”；我们要先创建一个产品，操作如下：

在在控制台中的”设备管理” -> “设备”中新建一个设备

通常设备在接入阿里云的时候需要一个”三元”认证，认证信息在创建的设备中可以看到。

物模型

物模型是阿里云对设备在云端的功能描述，包括设备的属性、服务和事件。

在首页”文档” -> 文档中心的”阿里云物联网平台” -> “操作指南”下的”设备接入” -> “使用协议自主接入” -> “基于 Alink 协议自主开发” 中的”设备属性、事件、服务”，中查看数据首发的一系列格式。
